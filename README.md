# ElvUI Enhanced Quest XP

![Version](https://img.shields.io/badge/version-1.00-blue)
![Dependency](https://img.shields.io/badge/requires-ElvUI-orange)

Плагин для **ElvUI**, который позволяет настроить отображение опыта за квесты с учётом серверных и персональных бонусов опыта.

## Описание

**Enhanced Quest XP** модифицирует полосу опыта ElvUI, учитывая:
- **Серверный множитель опыта** — если на сервере действует увеличенный рейт опыта за квесты (x2, x3, x5 и т.д.)
- **Персональный бонус опыта** — дополнительный процент опыта (от премиума, бонуса выходного дня и т.п.)

Это позволяет видеть более точную информацию о том, сколько опыта вы реально получите за выполнение квестов.

## Установка

1. Скачайте последнюю версию: [Releases](https://github.com/KataLoved/ElvUI_EnhancedQuestXP/releases/latest)
2. Распакуйте папку `ElvUI_EnhancedQuestXP` в директорию:
   ```
   Interface\AddOns\
   ```
3. Перезапустите игру или выполните `/reload`

## Настройка

1. Откройте настройки ElvUI
2. Найдите **Enhanced Quest XP**
3. Настройте параметры:
   - **Включить** — активирует/деактивирует модификацию
   - **Множитель сервера** — установите серверный рейт опыта (x1-x5)
   - **Процент бонусного опыта** — ваш персональный бонус (0-300%) (почему до 300%? - На всякий случай ¯\\_(ツ)_/¯)

## Зависимости
- `ElvUI`

## Значения по умолчанию

| Параметр | Значение |
|----------|----------|
| Включено | `true` |
| Множитель сервера | `x1` |
| Бонусный опыт | `0` |
